# 🎬 Netflix Clone

A responsive Netflix landing page clone built using HTML and CSS.
It replicates the basic UI of Netflix for learning and practice.

## 🚀 Features

* Responsive design
* Hero section with background banner
* Navigation bar
* Clean and modern UI

## 🛠️ Tech Stack

* HTML5
* CSS3

## ▶️ How to Run

1. Download or clone the repo
2. Open `vid53.html` in your browser

## 📁 Structure

* `vid53.html` – main page
* `style.css` – styling
* `assets/` – images & videos
